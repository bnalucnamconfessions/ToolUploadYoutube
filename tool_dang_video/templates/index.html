Hướng dẫn sử dụng

1

Chọn folder chứa video

Nhấn "Chọn Folder" và chọn thư mục chứa file video (mp4, mov, mkv, …) cần upload lên YouTube.

2

Cấu hình upload

Điền tiêu đề video (hoặc để trống), chọn Made for kids (Có/Không), Visibility (Riêng tư / Không công khai / Công khai) và tên file Excel để lưu link + trạng thái.

3

Bắt đầu upload và chọn tài khoản

Nhấn "Bắt đầu Upload" → chuyển sang trang chọn tài khoản (profile Chrome). Chọn profile để dùng; lần đầu có thể cần đăng nhập YouTube trong cửa sổ Chrome do tool mở.

4

Theo dõi tiến trình

Xem trạng thái upload, số thành công/thất bại và log chi tiết. File Excel được cập nhật realtime sau mỗi video (link + trạng thái), không cần đợi hết batch.

5

Tải file Excel

Sau khi xong (hoặc bất kỳ lúc nào), nhấn "Tải Excel" để tải file chứa link YouTube và trạng thái từng video đã upload.

Cấu hình Upload

Nhấn "Chọn Folder" để chọn folder chứa video
Để trống nếu không cần đặt tiêu đề; YouTube sẽ dùng tên file
